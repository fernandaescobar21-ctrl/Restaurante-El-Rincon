🍽️ El Rincón – Restaurante de Autor
¡Bienvenido a la web oficial de El Rincón! Este es un proyecto de sitio web moderno y elegante diseñado para un restaurante de cocina mediterránea. La página es totalmente interactiva, permitiendo a los usuarios explorar el menú y realizar reservas de mesa de forma sencilla.

✨ Características Principales
Diseño Elegante y Moderno: Uso de tipografías premium (Cormorant Garamond y Outfit) para una experiencia visual sofisticada.

Menú Interactivo: Sistema de pestañas (Tabs) para navegar entre Entradas, Principales, Postres y Bebidas.

Sistema de Reservas: Formulario funcional con validación y mensaje de confirmación dinámico.

Totalmente Responsivo: Adaptado para celulares, tablets y computadoras.

Navegación Fluida: Enlaces internos con desplazamiento suave (smooth scroll).

🛠️ Tecnologías Utilizadas
Para este proyecto se utilizaron herramientas estándar de desarrollo web para garantizar ligereza y velocidad:

HTML5: Estructura semántica del sitio.

CSS3: Estilos personalizados, animaciones y diseño adaptable (Flexbox/Grid).

JavaScript (ES6+): Lógica para el manejo del menú, el formulario de reservas y el menú móvil.

Google Fonts: Tipografías para una mejor estética.

📂 Estructura del Proyecto
El proyecto está organizado de la siguiente manera:

Bash
/
├── index.html    # Estructura principal del sitio
├── styles.css     # Todos los estilos visuales
├── script.js      # Lógica interactiva y manejo de datos
└── README.md      # Guía del proyecto
🚀 Cómo usar este proyecto
Clona o descarga este repositorio en tu computadora.

Asegúrate de tener los archivos index.html, styles.css y script.js en la misma carpeta.

¡Abre el archivo index.html en cualquier navegador (Chrome, Edge, Safari) y listo!

📸 Vista Previa de Secciones
1. Inicio (Hero)
Una bienvenida visual con botones de acción rápida para ver el menú o reservar directamente.

2. Menú
Un catálogo organizado donde el cliente puede filtrar los platos según su categoría.

3. Reservas
Un formulario intuitivo que simula el proceso de reserva, mostrando un mensaje de éxito al finalizar.

📝 Notas del Desarrollador
Este sitio fue creado pensando en la experiencia de usuario (UX). El diseño oscuro en la sección de reservas ayuda a separar visualmente las áreas de información de las áreas de acción.
